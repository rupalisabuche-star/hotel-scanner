# 🍽️ Hotel Scanner Project

This is a QR Code based Hotel Information System.

## 📌 Features
- Scan QR Code to open hotel website
- View hotel details
- Menu with price and photos
- Contact number & address
- Instagram link
- Attractive UI

## 🛠️ Technologies Used
- Python (Flask)
- HTML, CSS
- GitHub
- Render (Deployment)
